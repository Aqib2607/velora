# Velora — Design Document

## Theme System

Supports Light and Dark modes.

### Brand Colors

- Primary Purple: #6a329f
- Secondary Lavender: #b4a7d6
- Accent Gold: #f1c232
- Light Background: #ffffff
- Dark Background: #000000

---

## Typography

Font Stack:
"Segoe UI", Roboto, Helvetica, Arial, sans-serif

Scale:

- H1: 32px bold
- H2: 24px semi-bold
- Body: 14–16px regular

---

## Layout System

- 12-column responsive grid
- Max width: 1440px
- Mobile-first responsive
- Sticky header
- Sidebar dashboards for Seller/Admin

---

## Components

### Buttons

- Primary: Purple background (light)
- Lavender background (dark)
- Rounded 8px

### Cards

- 12px radius
- Subtle elevation
- Hover lift effect

### Forms

- Focus ring purple
- Error states red
- Floating labels

---

## Accessibility

- WCAG 2.1 AA
- 4.5:1 contrast ratio
- Keyboard navigable
